# TG soft-shill targets — verified public groups

Companion to `tg-soft-shill.md` (templates). This is the LIST of groups to post into.
All entries below have been preview-fetched at least once. Member counts as of 2026-05-19.
**Skip the "Base ecosystem maxi" groups entirely** — they ban explicit shills on sight and
the audience is brand-pure, not new-money. List below is degen / gem / multi-chain biased.

## Rules of thumb by category

- **Gem hunters / moonshots**: lead with the CA + chart link + 1-line mechanics hook. Pros + cons in same post. Effort-shills land, lazy ones get muted. Many require a min character count.
- **Multi-chain degen rooms**: lead with "new on Base" since most chasers are SOL/ETH-native and Base is the seasonal narrative. Drop chart, then one weird mechanic (permadeath, on-chain duels).
- **Shill-day / submission rooms**: respect the day-of-week and the hashtag if any. Wrong day = insta-ban.
- **NFT flipper rooms**: lead with art, then floor, then utility ($1 fight loop is the differentiator). Avoid "PFP" framing — say "playable 2k collection".
- **Pump/signal channels** (broadcast-only): NOT joinable for posting; their chat siblings (often `_chat` / `_talk` suffix) are where you actually drop. Always check for `_chat` variant.
- **Universal**: never repost same template inside 7 days, never post two links in one msg, read chat for 60s before dropping, reply to actual context where possible.

## Notes on what was deliberately skipped

- **Base-maxi rooms** (Base Ecosystem, Base Builders, etc.) — per brief, wrong-fit audience, ban-on-shill.
- **"Pump signal" private VIP rooms** (Binance Killers, DegenPump premium, Crypto Pump Club premium tiers) — paid-only, pump-and-dump adjacent, brand-toxic. Skipped on principle.
- **CryptoGemsCalls, DeFi_Million, cryptopumpclub, whaleclubchat, Crypto_Bull_Run_Group, CMCmoonshots, cmcofficialchat** — preview pages returned no member count / no description. Cannot verify they are real active joinable groups. Re-verify before targeting.
- **Crypto Insider** — exists (`@insider_crypto`) but is a paid-signals VIP funnel, not a community chat. Skipped.

## Recommended posting cadence

- Tier 1 (post first, highest signal): #1 CryptoMoonShots (write the long-form), #4 solanashill (short CA drop), #6 nftotc (listing-style), #7 NFTRoom (discussion bait).
- Tier 2 (after Tier 1 lands & you've got engagement screenshots): #2 SatoshiStreetBets, #9 dexscreenerchat (with real chart contribution), #3 satoshi_club (convo-bait).
- Tier 3 (opportunistic / milestone-driven only): #8 binancenfts, #11 CoinMarketCap.
- Max 2 groups per day, never identical text, log each post in `marketing/content/launch-receipts/` for future audit.

## Re-verify before each posting wave

Telegram groups churn fast. Before using this list, re-fetch the preview page of each target
to confirm it's still active and the member count hasn't collapsed. Dead groups burn the
template's freshness for no reach.
