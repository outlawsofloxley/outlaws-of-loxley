# TG soft-shill templates

Short, scroll-stopping, no-marketing-words posts for dropping into degen
TG groups (gem hunters, leverage rooms, NFT flippers, multi-chain alpha).
**NOT for the Base ecosystem rooms** — those treat shills as spam.

## Rules of engagement

- Pick ONE template per group, don't repost the same one twice in a week.
- Read the chat for 60 seconds before posting. If someone just shilled, wait.
- Reply to relevant context where possible (someone says "low cap" → drop a chart variant). Cold drops get scrolled past.
- Always include a link, never both X + site + chart in the same post.
- If a mod or regular asks a follow-up, answer with substance and DON'T re-paste the template. Get a convo going.
- If group rules say "shills only on Sundays" / "use #shill tag" / etc., follow them. One ban kills the channel forever.

## What NOT to post

- ❌ "100x soon" / "gonna moon" / "easy 10x"
- ❌ "presale spots dm me" (there is no presale, ever)
- ❌ "guaranteed" anything
- ❌ Long pastas with emojis and rocket ships
- ❌ The same template twice in 24h in any single group
- ❌ Cross-promoting other coins to legitimize ours
- ❌ Tagging accounts that didn't ask

## After posting

- Check back in 10 min. If someone replies, answer them as a human (game-first, never "ser thanks for ape").
- If 0 reactions in 30 min, that group is dead, mark it for de-prioritization.
- Log every post (group, template letter, time, reactions) in a simple sheet so we don't blast the same room.
